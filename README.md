# Digital Business Card
This is a solution to the Digital Business Card solo project on Scrimba. <hr>

### Requirements

- Build from scratch
- Fill in your own information
- Separate components for:
  - Info (photo, name, buttons, etc.)
  - About
  - Interests
  - Footer (social icons) <hr>
